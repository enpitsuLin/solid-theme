# solid-themes

[![pnpm](https://img.shields.io/badge/maintained%20with-pnpm-cc00ff.svg?style=for-the-badge&logo=pnpm)](https://pnpm.io/)

An abstraction for themes in your solidjs app.

## Quick start

Install it:

```bash
npm i solid-themes
# or
yarn add solid-themes
# or
pnpm add solid-themes
```
